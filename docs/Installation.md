## Requirements

- Java 21
- Paper or Folia 1.21+
- PacketEvents 2.12.1+
- PlaceholderAPI if you use placeholders or requirement checks

## Steps

1. Stop the server.
2. Put `PageLore.jar` in the `plugins` folder.
3. Put `packetevents` in the `plugins` folder.
4. Add PlaceholderAPI if your lore uses `{papi:...}` or `{check:...}`.
5. Start the server once so PageLore creates its files.
6. Edit `plugins/PageLore/config.yml`.
7. Run `/pagelore reload` or restart the server.

## Permissions

`pagelore.admin` allows access to `/pagelore reload`.
